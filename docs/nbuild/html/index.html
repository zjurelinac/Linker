App.py¶

App configuration file and initialization file

The Flask app object is created here, as well as the peewee db database. This module also contains app configuration options, such as database name, a not-so-secret key and debug status

Run.py¶

Runserver script

Imports all the application parts, creates those tables which are missing and then starts the application.

Utils.py¶

Module containing utility functions

utils.hashfunc(str)¶

Returns a hash value of a given string

Takes a string and returns its SHA512 hash value, encoded in base64

utils.markup_to_html(str)¶

Transforms a simple markup string into html

Supported are: bold as ‘bold part‘ and italic as ‘italicized part‘

utils.pretty_date(time=False)¶

Returns a string containing a human-readable date

Get a datetime object or a int() Epoch timestamp and return a pretty string like ‘an hour ago’, ‘Yesterday’, ‘3 months ago’, ‘just now’, etc
